## 🏗 安装

```
// npm
npm install --save --dev @walrus/cli @walrus/plugin-commitlint

// yarn
yarn add --dev @walrus/cli @walrus/plugin-commitlint
```

建议直接安装 [@walrus/preset-lint](https://github.com/walrusjs/plugins/tree/master/packages/preset-lint)
